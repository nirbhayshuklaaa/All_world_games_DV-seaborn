This project focuses on data visualization of global sports/games datasets using Python libraries. The aim is to analyze patterns, trends, and insights from games played across the world through interactive and informative visualizations.

📊 Objectives
Understand distribution of different sports worldwide
Analyze frequency and popularity of games
Identify trends using graphical representations
Build strong data visualization skills using Python

🛠️ Tools & Technologies Used
Python
Jupyter Notebook
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib – Basic plotting
Seaborn – Advanced visualization
📁 Dataset - Games.csv

The dataset contains information about:

Game/Sport Name
Country/Region
Frequency/Count of games

📈 Visualizations Created
Some of the key visualizations in this project include:

📌 Bar Plot – Top sports by frequency
📌 Histogram – Distribution of values
📌 Scatter Plot – Relationship between variables
📌 Count Plot – Occurrence of categorical data
📌 pie chart -male vs female and season vs games ratio



📌 Key Insights
Certain sports dominate globally in terms of frequency
Data distribution shows skewness in some categories
Visualization helps in better understanding of large datasets
